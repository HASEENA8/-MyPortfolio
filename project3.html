Rainfall Prediction Using ML

This is a machine learning project that predicts the chance of rainfall based on various weather parameters using historical data.

Features:

  • Uses historical datasets to train ML models
  • Real-time prediction UI (Tkinter & Flask)
  • Classification for Rainfall Severity
  • Live weather integration (optional)

Technologies Used:

Python, Pandas, Scikit-Learn, Flask, Tkinter
