🧠 Cyber Steganography Tool

A desktop Python application that hides and reveals secret messages inside images using AES encryption and LSB steganography.

🧩 Features

  • 🖼️ Drag-and-drop image support (.png/.bmp)
  • 🔐 AES-256 encryption with password
  • 🕵️‍♂️ LSB-based steganography using PIL
  • 🧾 View and export message history
  • 📤 Export logs as CSV
  • 🎨 Stylish GUI using Tkinter

📸 Screenshot

🔧 Technologies Used

Python (Tkinter, PIL), AES Encryption (cryptography), LSB Bit Manipulation, CSV Logging

🚀 How to Run


pip install pillow cryptography
python stego_tool.py

📂 Output Files

  • history.csv – Log of activity
  • *.png – Modified image with message
  • Optional: report.pdf

📌 Notes

  • Only PNG or BMP images are supported
  • Password required to decrypt correctly
  • Message size depends on image resolution
