🛡️ XSS Vulnerability Scanner

A Python Flask-based web application that helps detect reflected XSS (Cross-Site Scripting) vulnerabilities in web apps by injecting common payloads and analyzing responses.

🔍 Features

  • 🔗 URL input for scanning
  • 💣 Injects multiple XSS test payloads
  • 🧪 Detects reflected XSS vulnerabilities
  • 📥 Export results as CSV or PDF
  • 📊 Stylish dashboard with results
  • ⚡ Easy to use and lightweight

📸 Screenshot

⚙️ Tech Stack

Backend: Python, Flask
Frontend: HTML, CSS (Bootstrap 5)
Libraries: requests, pandas, reportlab, flask

🚀 Getting Started


git clone https://github.com/yourusername/xss-vulnerability-scanner.git
cd xss-vulnerability-scanner
pip install -r requirements.txt
python app.py

Then open your browser and visit: http://127.0.0.1:5000/
